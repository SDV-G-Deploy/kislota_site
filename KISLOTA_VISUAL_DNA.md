# KISLOTA Visual DNA

## 1) Brand formula
**Editorial discipline × acid signal punctuation × operational clarity.**

Kislota is not “pretty chaos”. It is a control-room publication with voltage.

## 2) Palette philosophy
- **Base = dark, quiet, readable.** Surfaces carry structure.
- **Accents = neon, sparse, intentional.** Color means priority/state, not decoration.
- **One frame, one dominant accent family.** Others are secondary punctuation.

## 3) Intensity scale (0–4)
- **0 — Silent:** article reading, long-form pages.
- **1 — Calm signal:** default UI chrome, metadata, dividers.
- **2 — Editorial push:** homepage modules, cards, section anchors.
- **3 — Acid punctuation:** hero word, ticker hits, 1–2 hotspots max per viewport.
- **4 — Alarm mode (rare):** urgent events only, short-lived.

Default product mode: **1–2**. Homepage peaks at **3** in controlled nodes.

## 4) Weirdness budget
- Keep weirdness in **10–15% of visible UI** on home.
- Max **2 strange hotspots** per screen: never all modules at once.
- If a block is loud, adjacent blocks must be calm.

## 5) Accent grammar
- **Psychic/teal** = lead signal, trust, navigation spine.
- **Alien/blue** = informational flow.
- **Threat/red** = urgency/risk.
- **Acid/lime + violet/magenta** = anomaly/feature punctuation only.
- Use glow as **edge emphasis**, not full-area wash.

## 6) Page role split
- **Home:** highest contrast + kinetic editorial rhythm.
- **Hero:** one dominant narrative + strongest accent budget.
- **Category/sections:** organizational, semantically colored, less glow.
- **Featured block:** premium framing, one controlled neon edge.
- **Article page:** quieter, low pulse, text-first, near scale 0–1.

## 7) Hard rule
**Structure > chaos.**

Hierarchy, spacing, and reading order are non-negotiable. Accent never destroys scanning.

## 8) “Scroll-stopping but not exhausting”
- Stop: one sharp visual hook per module.
- Not exhausting: no constant flicker, no blanket glow, no multi-color noise stack.
- User should feel “curated signal”, not “rave poster wall”.
